SED-ML examples

Below are several examples of SED-ML files. Each example is provided as a COMBINE archive, which contains both the example SED-ML file and all model and data files involved in the SED-ML file.

For each example, the table below also lists simulation tools which have been verified to be able to execute the example.

Note, because SED-ML files typically describe simulations involving specific model formats and simulation algorithms, each SED-ML file can only be executed by a subset of simulation tools. Information about the capabilities of simulation tools to execute SED-ML files involving specific formats and algorithms is available at BioSimulators. In addition, runBioSimulations provides a web application which can recommend specific tools for executing particular SED-ML files in COMBINE archives.

Additional examples

BioModels database
Database of models which includes SED-ML files for many entries. Entries with SED-ML files can also be obtained as COMBINE archives.

BioSimulators test suite
Test suite for checking that simulation tools execute SED-ML files and COMBINE archives consistently with the specifications of these formats. The test suite uses a collection of curated COMBINE archives, each of which includes SED-ML files. These examples involve multiple model languages, including BNGL, SBML, SBML-fbc, and SBML-qual. To test simulation tools that involve a broad range of model languages, modeling frameworks, and simulation algorithms, the test suite can generate synthetic COMBINE archives and SED-ML files that are appropriate for testing a particular simulation tool (i.e., SED-ML files that involve model formats and simulation algorithms that the simulator can execute). The test suite can also save these synthetic archives to files for further inspection.

JWS Online simulation database
Database of simulation experiments which can be exported to SED-ML files and COMBINE archives.
