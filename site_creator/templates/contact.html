The SED-ML community

SED-ML is a community project that is under continuous development.
The SED-ML community comprises simulation software developers and modelers from the different communities applying computational methods on biology, medicine, neuroscience etc.

sed-ml-discuss (mailing list and forum)
The official mailing list for the SED-ML project is sed-ml-discuss. The mailing list is free for everyone to join. It is the place where all issues regarding the SED-ML language are being discussed. Announcements of SED-ML related events and support issues are posted on SED-ML disucss. Requests for new SED-ML features should be made through the mailing list also. Another important function of the mailing list is the voting for SED-ML editors, as well as for SED-ML proposals. Write and subscribe to the google group or write an email to sed-ml-discuss AT googlegroups.com.
Github project site
We are in the process of migrating all SED-ML information to github project site. Until the migration is complete please use the sourceforge project site below to access information.
Sourceforge project site
The SED-ML Sourceforge project site hosts all code related to the SED-ML language, including the SED-ML XML Schema, the SED-ML specification document, SED-ML feature requests and proposals for SED-ML extensions.
SED-ML editors
The SED-ML editors are being elected by the members of the sed-ml-discuss mailing list for a period of three years. SED-ML editors are responsible for hosting all material relevant to SED-ML, coordinating the SED-ML development, maintaining the SED-ML sourceforge website and the SED-ML specification document.
SED-ML meetings
SED-ML is part of the COMBINE effort. The SED-ML community meets twice a year in person, at the annual COMBINE meeting and at the annual HARMONY meeting. In addition there are regular online meetings.

SED-ML is a community effort, involving software developers of various simulation tools and model representation formats. Our effort values all opinion and contributions. If you have a question, want to discuss the evolution of SED-ML, want to voice an opinion, write and subscribe to sed-ml-discuss on google groups.
