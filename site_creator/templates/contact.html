Contact Us

SED-ML is a community project that is under continuous development, involving software developers of various simulation tools and model representation formats. Our effort values all opinion and contributions. If you have a question, want to discuss the evolution of SED-ML, want to voice an opinion, write and subscribe to sed-ml-discuss.

SED-ML is part of the COMBINE effort. The SED-ML community meets twice a year in person, at the annual COMBINE meeting and at the annual HARMONY meeting. In addition there are regular online meetings.

Forum
https://groups.google.com/forum/#!forum/sed-ml-discuss
The official mailing list for the SED-ML project is sed-ml-discuss. The mailing list is free for everyone to join. It is the place where all issues regarding the SED-ML language and SED-ML features are being discussed. Announcements of SED-ML related events and support issues are posted on SED-ML discuss.
Another important function of the mailing list is the voting for SED-ML editors, as well as for SED-ML proposals. Write and subscribe to the google group or write an email to sed-ml-discuss@googlegroups.com.
Repository
https://github.com/SED-ML
The SED-ML repository hosts all code related to the SED-ML language, including the SED-ML XML Schema, the SED-ML specification document, SED-ML feature requests and proposals for SED-ML extensions.
Email
sed-ml-editors@googlegroups.com
If you want to directly contact the SED-ML editors, please send a mail to sed-ml-editors@googlegroups.com.
Issue Tracker
https://www.pivotaltracker.com/n/projects/2074517
To report a bug or record an issue with the current specification, please submit an item to the issue tracker L1V2 tracker. For feature requests for future specifications, please submit an item to the feature request tracker.
